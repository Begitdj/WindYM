# WindYM

Минималистичный сторонний Android-клиент для Яндекс Музыки.
Поддерживает Android 4.1+ (API 16). 

> **Примечание:** Весь код этого приложения был полностью написан с помощью ИИ (нейросети).

## Особенности
- Лёгкий и быстрый интерфейс
- Полная поддержка "Моей волны", избранного, поиска и плейлистов
- Скачивание треков напрямую в память устройства
- Системный эквалайзер
- Воспроизведение в фоне (через ExoPlayer) с интеграцией в шторку уведомлений

## Как пользоваться
Для использования потребуется ваш персональный токен Яндекс Музыки:
1. Перейдите по ссылке: `https://oauth.yandex.ru/authorize?response_type=token&client_id=97fe03033fa34407ac9bcf91d5afed5b`
2. Войдите в свой Яндекс ID
3. Скопируйте строку после `access_token=` из URL браузера
4. Вставьте этот токен при первом входе в приложение

## Сборка
Проект использует стандартный Gradle:
```bash
./gradlew assembleRelease
```
